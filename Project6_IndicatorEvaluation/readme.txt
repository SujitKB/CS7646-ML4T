#README

The submissions include:
1. indicators.py
2. marketsimcode.py
3. TheoreticallyOptimalStrategy.py

Part 1 of the project can be run by executing indicators.py.
Part 2 of the project can be run by executing TheoreticallyOptimalStrategy.py.
The “main” code in TheoreticallyOptimalStrategy.py call marketsimcode.py functions and also implements API call for TheoreticallyOptimalStrategy.
